# DBMS Lab Experiments – SQL 

## Overview
This repository contains SQL laboratory experiments for the course **Introduction to Database Systems**.  
The experiments are implemented using **Oracle SQL / Oracle APEX** and follow the prescribed university lab manual format.

Each experiment includes:
- Problem statements as per the lab manual
- SQL queries
- Step-by-step explanations
- Output tables
- Final results

---

## Tools & Technologies
- Oracle SQL
- Oracle APEX
- Relational Database Management System (RDBMS)

---

## Experiments Included

### Experiment 1: DDL – Data Definition & Table Creation
- Creation of `PRODUCTS`, `CUSTOMERS`, and `ORDERS` tables
- Usage of primary keys and foreign keys
- Table structure verification using `DESC`
- Output tables included

### Experiment 2: DML & DQL – Insert, Select, Update & Delete
- Insertion of multiple records using `INSERT ALL`
- Data retrieval using `SELECT`
- Record modification using `UPDATE`
- Deletion of inactive records using `DELETE`
- Output tables shown after each operation

### Experiment 3: Aggregate Functions with GROUP BY and HAVING
- Creation of `SALES_DATA` table
- Insertion of sales records
- Use of aggregate functions:
  - COUNT
  - SUM
  - AVG
- Queries using `GROUP BY`, `HAVING`, and `ORDER BY`
- Tabular outputs for all queries
